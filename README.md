# SSO

**SSO Starter** basic implementation of use the library [sso-starter](https://github.com/sysnormal1/sysnormal-spring-boot-starter-sso.git). 

---
## ⚙️ Configuration

> ⚠️ if correct configured on application.yml, the database is auto-create on startup.
---


## 🧬 Clone the repository

To get started locally:

```bash
git clone https://github.com/sysnormal1/sso-service.git
cd sso-service
mvn clean package
```

## 🔧 Build and Local Test

```bash
mvn clean package
```
